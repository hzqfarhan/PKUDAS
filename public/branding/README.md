# e-Dent Branding & Assets

This folder contains the branding assets used across the e-Dent application.

## Where to Replace Placeholders

**1. Main Background Image**
- Located at: `public/images/backgrounds/uthm-clinic-bg-placeholder.jpg`
- Usage: Applied as the subtle full-page background in the public app shell.
- Recommendation: Replace with a high-quality, calm photo of the UTHM clinic or a related clean medical environment (preferably a light/airy image).

**2. UTHM Logo**
- Located at: `public/branding/uthm-logo-placeholder.svg`
- Usage: Displayed in the sidebar and navigation header.
- Recommendation: Use a transparent PNG or SVG of the official UTHM logo.

**3. e-Dent Logo**
- Located at: `public/branding/e-dent-logo-placeholder.svg`
- Usage: Main application logo used alongside the name.
- Recommendation: Use an SVG icon that matches the e-Dent medical theme (e.g., a modern tooth or clinic crest).

**4. PWA Icons**
- Located at: `public/icons/pwa/icon-192.png` and `public/icons/pwa/icon-512.png`
- Usage: These icons are used when users install the PWA on their devices.
- Recommendation: Ensure they are square PNGs matching the e-Dent teal branding `(#05384C)`.
